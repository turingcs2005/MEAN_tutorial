Welcome to the MEAN stack tutorial!

This tutorial provides a comprehensive guide to building a full-stack application using MongoDB, Express, Angular, and Node.js. As of November 2025, this tutorial reflects the best practices and commonly used tools in the MEAN stack ecosystem.

The software versions used to create this tutorial are listed below.

Runtime stack: software required to run your app
  • MongoDB 8.2
  • Express 5.1
  • Angular 21.0
  • Node.js 24.11
 Build-time stack: software required to build your app
  • Angular CLI 21.0
  • Node Package Manager (npm) 11.6
  • TypeScript 5.9
 Developer tools: software required to enhance development
  • Visual Studio Code
  • Git 2.51
  • Node Version Manager (nvm) 0.40
  • (Optional) Fira Code

Prerequisites for following this tutorial:

  • Basic understanding of JavaScript and TypeScript
  • Familiarity with web development concepts (HTML, CSS/SCSS, HTTP)
  • Basic knowledge of command line operations
  • Familiarity with Visual Studio Code
