Welcome to the MEAN stack tutorial!

With AI, coding has become more efficient and enjoyable. This tutorial provides a comprehensive guide to building a full-stack application using MongoDB, Express, Angular, and Node.js.

As of November 2025, this tutorial reflects the best practices and commonly used tools in the MEAN stack ecosystem.

The following software programs are required to set up your development environment. Use an AI assistant to help install them on your system.

Runtime stack: these software are required to run your MEAN app.

  • MongoDB 8
  • Express 5
  • Angular 21
  • Node.js 24
  • TypeScript 5

Build-time stack: these software are required to build your MEAN app.

  • Angular CLI 21
  • Node Package Manager (npm) 11
  • TypeScript Compiler 5

Developer tools: these tools make development faster and smoother.

  • Visual Studio Code and plugins:
    • Angular Language Service
    • AI plugin such as GitHub Copilot
    • MongoDB for VS Code
    • (Optional) Vim
  • Git
  • (Optional) Node Version Manager (nvm) and packages
  • (Optional) Fira Code
