Icons

You can search for Material icons and symbols here at Google. You may also ask AI to make icon/symbol suggestions for your project.

1. Material Icons and Material Symbols

Classic Material Icons are a subset of Material icons and Symbols. By default, the index.html file should have already imported them. (line 1 below) If classic Material icons aren't enough and you want to use Google's new Material symbols, import them by adding a 2nd line to your index.html. (line 2 below)

Using classic Material icons or the new Material symbols in your Angular app is straightforward:

home classic Material icon for 'home'. <mat-icon>home</mat-icon>
home new Material symbol for 'home'. <mat-icon fontSet="material-symbols-outlined">home</mat-icon>

2. Custom SVG Icons

Often we need custom icons for branding. Angular Material allows you to register custom SVG icons and use them just like the built-in icons.

  1. Place your custom SVG file (e.g. lobster.svg) in /public/icons/ directory (or whatever directory you prefer)
  2. Import MatIconModule (which you should have already done in your component, as it is needed for Material icons and symbols)
  3. Register your custom SVG icon in your component's TypeScript file using MatIconRegistry and DomSanitizer.
  4. Use the registered icon in your template with the svgIcon attribute.
Custom SVG icon for 'lobster'. <mat-icon svgIcon="lobster"></mat-icon>
